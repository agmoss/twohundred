## watchful [![NPM Package](https://img.shields.io/npm/v/watchful)](https://www.npmjs.com/package/watchful)

watchful is a nodejs CLI for checking the HTTP status of Urls

## Installation

```bash
npm install watchful -g
```

## Usage

```bash
watchful
```

### Help

```bash
watchful --man
```

### Run Tests
```bash
npm test
```
